Styling By CSS

Inline CSS

CSS helps make web page attractive and colorful.Inline CSS is used when we want to style a specific element directly inside HTML tag.

Internal CSS

Internal CSS is written inside the style tag of an html document.It helps to maintain consistency across multiple web pages and keeps the code clean and clear.

Class Selector

Class Selector allow us to apply the same style to multiple elements.

External CSS

External CSS is stored in a separate file and linked to html page.This makes the code oragnized, reusable,and easier to manage.

ID Selector

ID Selector are used to style a unique element on a page.Since it should be used only once, it is helpful for highlighting important content.
